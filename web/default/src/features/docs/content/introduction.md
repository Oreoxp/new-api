# 介绍

OmniAPI 是一个 AI 模型聚合网关（中转站）。它把众多上游大模型提供商的接口、密钥与计费，统一收束到**一个端点**之后，让你用同一套 OpenAI 兼容协议访问全部模型。

## 它能做什么

- **一个端点，接入全部**：一次对接即可调用 OpenAI、Claude、Gemini、DeepSeek、Qwen 等众多模型，无需为每家单独接入。
- **OpenAI 兼容**：沿用 `/v1/chat/completions` 等标准接口，已有代码通常只需把 `base_url` 指向 OmniAPI 即可。
- **统一计量与计费**：所有调用集中记账，用量、余额、日志一处可查。
- **稳定中转**：多渠道负载均衡与秒级故障转移，上游波动时尽量保证你的请求平稳。

## 适合谁

- 想用一套代码、一把密钥访问多家模型的开发者；
- 需要集中管理团队用量、额度与权限的团队；
- 希望快速试用前沿模型、无需逐家注册的个人用户。
