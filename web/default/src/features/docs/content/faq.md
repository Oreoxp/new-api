# 常见问题

## 和直接用官方接口有什么区别？

OmniAPI 用同一套 OpenAI 兼容协议聚合多家模型：一把密钥、一个基址即可切换不同模型，并集中管理用量与计费。已有代码通常只需改 `base_url`。

## 需要改动很多代码吗？

通常不需要。把 `base_url` 指向 `https://omniapi.top/v1`、把密钥换成 OmniAPI 的密钥即可，其余请求 / 响应结构与 OpenAI 一致。

## 支持流式吗？

支持。在请求体加 `"stream": true`，详见[流式输出](/docs/streaming)。

## 报错 401 / 鉴权失败怎么办？

检查 `Authorization: Bearer sk-...` 头是否正确、密钥是否被禁用或删除。详见[错误码](/docs/errors)。

## 提示模型不存在？

该模型名当前不可用。请改用[模型列表](/docs/models)中确实存在的模型名。

## 余额不足 / 被限流？

请到控制台「钱包」充值，或在「使用日志」核对消耗；如需更高并发 / 额度，可在控制台调整或联系管理员。

## 密钥泄露了怎么办？

立即到控制台「API 密钥」删除或禁用该密钥，并创建新的。密钥等同账户凭证，请勿提交到公开仓库。
