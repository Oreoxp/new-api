# 模型与计费

## 查看可用模型

可用模型会随上游能力持续更新。获取当前列表有两种方式：

- 在控制台的「模型 / 定价」页面浏览；
- 调用接口查询：

```bash
curl https://omniapi.top/v1/models \
  -H "Authorization: Bearer $OMNI_API_KEY"
```

调用时把请求体的 `model` 字段设为列表中的模型名即可。

## 计费方式

- **按量计费**：按每次请求实际消耗的 token 计量，响应中的 `usage` 字段给出本次输入 / 输出 token 数。
- **不同模型不同价**：各模型的单价不同，以控制台「定价」页面公布为准。
- **余额与用量**：可在控制台「钱包」查看余额、在「使用日志」查看每次调用的明细。

## 额度与限流

- 账户余额不足时请求会被拒绝，请及时充值。
- 为保障稳定，系统可能对单位时间的请求数 / 并发做限制；如需更高额度可在控制台调整或联系管理员。

> 价格、可用模型与限流策略均以控制台实际展示为准，本页仅说明机制。
